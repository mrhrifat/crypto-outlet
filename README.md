### Cyrpto Outlet
Cypto Outlet is a React Application that provides a platform for know crypto currencies, crypto exchanges and crypto news. 

### Technologies Used
1. Programming Language & Others
    - HTML
    - CSS
    - JavaScript
    - ReactJS
    - Redux
2. Tools & Libraries
    - Ant Design
    - Axios
    - Chart.js
    - Millify
    - Moment.js
3. API
    - Coin Raking
    - Bing News Search

### Features
- Crypto Currencies
- Crypto Exchanges
- Crypto News
